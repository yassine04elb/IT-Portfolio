# Freelance IT Support Technician

## Vaardigheden aangetoond
- Diagnose en oplossen van hardware/software problemen
- Wifi- en netwerkproblemen troubleshooten
- Malware verwijdering en beveiliging
- Data recovery en backup-advies
- Wachtwoord recovery
- Klantgericht communiceren met niet-technische gebruikers

## Tools gebruikt
- Windows Troubleshooting (BSOD, Startup Repair, System Restore)
- Malwarebytes, Windows Defender
- Recuva, TestDisk
- Remote Desktop, AnyDesk
